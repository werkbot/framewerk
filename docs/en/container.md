# Container

The container component is an optional class which is used to set a consistent max width of the website.

You can set the width in the theme: 
`layoutContainerWdth: 1366px;`

Example:
```
<div class="header">
    <div class="fw-container">
        ...
        CONTENT
        ...
        ...
    </div>
</div>
<div class="content">
    <div class="fw-container">
        ...
        CONTENT
        ...
        ...
    </div>
</div>
```
